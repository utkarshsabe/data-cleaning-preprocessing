# Data Cleaning and Preprocessing

## Objective

Create a data cleaning and preprocessing pipeline for a dataset containing missing values, duplicates, and inconsistent data.

## Dataset

The dataset contains student information with:

* Missing values
* Duplicate records
* Inconsistent text formatting

## Technologies Used

* Python
* Pandas

## Steps Performed

1. Loaded the CSV dataset.
2. Identified missing values.
3. Filled missing values using mean imputation.
4. Removed duplicate records.
5. Standardized text formatting.
6. Saved the cleaned dataset.

## Files

* student_data.csv – Original dataset
* data_cleaning.py – Python script
* cleaned_student_data.csv – Cleaned dataset
* README.md – Project documentation

## Result

The dataset was successfully cleaned and prepared for analysis and machine learning tasks.
